# Image Viewer

ローカルフォルダ内の画像を閲覧するためのシンプルなウェブベースの画像ビューアです。

## 概要

このツールは、ブラウザ上で動作する軽量な画像ビューアです。指定したフォルダ（サブフォルダ含む）内の画像を読み込み、フォルダごとに整理して表示します。

## 主な機能

- **フォルダ選択**: ローカルのフォルダを一括で選択し、画像を読み込みます。
- **対応フォーマット**: `.jpg`, `.jpeg`, `.png`, `.webp` 形式の画像に対応しています。
- **画像表示**:
  - 1ページにつき最大3枚の画像を並べて表示します。
  - 画像の下にはファイル名が表示されます。
- **自動ソート**:
  - フォルダごとに画像をグループ化します。
  - フォルダ内の画像は「更新日時が新しい順」に表示されます。
  - サブフォルダ自体も、中に含まれる最新の日付順に並び替えられます。
- **スライドショー**:
  - 自動再生機能（PLAYボタン）があります。
  - 再生速度をスライダーで調整可能です（1秒〜10秒間隔）。

## 使い方

1. `viewer.html` をウェブブラウザ（Chrome, Edge, Firefoxなど）で開きます。
2. 画面左上の「フォルダを選択」ボタンをクリックします。
3. 画像が保存されているフォルダを選択し、「アップロード」（または「選択」）をクリックします。
   - ※ブラウザの仕様により「アップロードしますか？」という確認が出ることがありますが、ファイルはローカルで処理され、外部サーバーに送信されることはありません。
4. 画面下の「前へ」「次へ」ボタンでページを切り替えて閲覧します。
5. 右上の「PLAY」ボタンでスライドショーを開始・停止できます。
